# Species: Benthic

**Classification:** Client Application / Viewer
**Role:** The Suitor

This directory contains the DNA required to reconstruct a Benthic Metaverse Client.

## Subspecies
* **0.1.0**: The experimental Rust-based client.

## Artifacts
* `acquire.sh`: Clones the required repositories (Client, Mesh, Serde).
* `incubate.sh`: Compiles the Specimen (Vanilla).
* `adapt_deepsea.patch`: Transforms the GUI client into a Headless Client for console operation.
